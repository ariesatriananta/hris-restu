# Knowledge Base — Payroll

> Status dokumen: kerangka aktif. Modul Payroll masih placeholder; isi dan perbarui saat implementasi dimulai.

## Status saat ini

Menu Payroll tersedia sebagai navigasi, tetapi periode, simulasi, approval, closing, snapshot, slip gaji, dan API bisnis belum diimplementasikan.

## Ruang lingkup yang direncanakan

Payroll borongan dan fondasi payroll bulanan, periode payroll, perhitungan/simulasi, approval berjenjang, closing immutable, snapshot nilai transaksi/tarif, histori, dan slip gaji.

Status schema yang direncanakan untuk periode adalah `DRAFT`, `CALCULATED`, `APPROVED`, `CLOSED`, dan `CANCELLED`. Status ini belum menjadi fitur operasional sampai Eksekusi Payroll dilakukan.

## Aturan pembaruan

Saat modul dibangun, jelaskan formula, sumber data, cut-off, status, approval, koreksi, closing, dampak pajak/potongan, akses per role, audit, dan contoh skenario operasional dalam dokumen ini.
